# Starter kit for p5.js with TypeScript and Vite

1. Clone this repo

```bash
git clone https://github.com/JuanCarlosllh/P5-TS-Vite-Starter
```
```bash
cd P5-TS-Vite-Starter
```

2. Install dependencies

```bash
pnpm install
```

3. Run the project
```bash
pnpm dev
```

Now you can edit the sketch file at ```./src/sketch.ts```
